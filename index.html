What is JavaScript?

JavaScript is a high-level, interpreted programming language that conforms to the ECMAScript specification. It has curly-bracket syntax, dynamic typing, prototype-based object-orientation, and first-class functions.

As a multi-paradigm language, JavaScript supports event-driven, functional, and imperative programming styles. It has APIs for working with text, arrays, dates, regular expressions, and the DOM (Document Object Model), but does not include any I/O, such as networking, storage, or graphics facilities. Instead, these functionalities are handled by the host environment in which JavaScript is embedded.

JavaScript engines were originally used only in web browsers, but they are now core components of other software systems, most notably servers and a variety of applications. The most popular runtime system for this usage is Node.js.

Client-side JavaScript extends the core language by supplying objects to control a browser and its Document Object Model (DOM). For example, client-side extensions allow an application to place elements on an HTML form and respond to user events such as mouse clicks, form input, and page navigation.

JavaScript's dynamic capabilities include runtime object construction, variable parameter lists, function variables, dynamic script creation (via `eval`), object introspection (via `for...in`), and source-code recovery (JavaScript functions store their source text and can be retrieved through `toString()`).

Variable Declarations

var

  • Global scope - can be seen outside of block
  • Can be redeclared and redefined
  • Hoisted - will be declared at top of scope but will not be defined

    • let

  • Block scope - can only be seen inside of block
  • Can be redefined but not redeclared
  • No Hoisting

    • const

  • Block scope - can only be seen inside of block
  • Cannot be redefined or redeclared
  • No Hoisting

    • Variable Types

  • Number - 42, -7, 3.14
  • String - "Hello World", "42"
  • Bigint - [-2^(2^53 - 1) - 2^(2^53 - 1) - 1]
  • Boolean - True / False
  • Null - let num = null;
  • Undefined - let num;
  • Symbol - Unique and Immutable. Ex - const sym = Symbol('description');
  • Object - Let obj = {x, y, z}

    • Operators

  • Arithmetic operators: +, -, *, /, %, ++, --
  • Assignment operators: =, +=, -=, *=, /=, %=
  • Comparison operators: == (), ===, !=, !==, >, <, >=, <=
  • Bitwise operators: &, |, ^, ~, <<, >>, >>>
  • Type operators: typeof, instanceof
  • String operators: +
  • Conditional operators: ?:
  • Unary operators: delete, void, typeof, +, -
  • Relational operators: in, instanceof
  • Grouping operators: ( )
  • Precedence operators: ()
  • Logical operators: &&, ||, !
  • Ternary operator: condition ? expr1 : expr2
